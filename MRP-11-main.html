üì± Complete Termux Setup Guide

1

Install Termux

Download and install Termux from F-Droid or Google Play Store. F-Droid version is recommended for latest updates.

2

Initial Setup

Open Termux and run the initial setup commands:

pkg update && pkg upgrade -y
pkg install php curl wget git python openssh
3

Storage Access

Grant Termux access to your device storage:

termux-setup-storage

Allow the permission when prompted.

4

Download Files

Navigate to shared storage and download the MRP files:

cd /storage/emulated/0/
wget https://your-server.com/MRP-11.html
wget https://your-server.com/response.php

Or transfer files manually to your Downloads folder.

5

Start Server

Start the PHP server to host your files:

php -S 0.0.0.0:8080

Server will start and display "Development Server started"

6

Find Your IP

In a new Termux session, find your phone's IP address:

ip route get 1.1.1.1 | grep -oP 'src \K\S+'

Note this IP address for network access.

7

Test Access

Test server access from another device:

http://[YOUR-PHONE-IP]:8080

You should see this landing page.

üîß Troubleshooting Guide

‚ùó

Server Won't Start

  • Port in use: Try php -S 0.0.0.0:8081
  • Permission denied: Check file permissions
  • PHP not found: Reinstall with pkg install php
‚ùó

Can't Access from Other Devices

  • Wrong IP: Re-check phone IP address
  • Firewall: Disable phone firewall temporarily
  • Network isolation: Ensure devices on same network
  • Server binding: Use 0.0.0.0:8080 not localhost
‚ùó

Receipt Printing Issues

  • Printer not found: Check Bluetooth pairing
  • Print failure: Verify printer supports URL schemes
  • Format issues: Test with simpler receipt data
